# FilmWorld - افزونه دانلود و تماشای فیلم و سریال

افزونه FilmWorld یک سیستم کامل مدیریت دانلود و استریم فیلم و سریال برای وردپرس است. این افزونه با زبان فارسی طراحی شده و از درگاه پرداخت زرین‌پال پشتیبانی می‌کند.

## ویژگی‌ها

- **پست تایپ سفارشی** برای فیلم‌ها و سریال‌ها
- **تکسونومی‌های اختصاصی:** ژانر، کشور، وضعیت دوبله
- **صفحه اصلی** با جدیدترین فیلم‌ها، سریال‌ها و فیلم‌های دوبله
- **صفحه تکی فیلم** با پوستر، اطلاعات، جدول دانلود، پلیر آنلاین و فیلم‌های مشابه
- **صفحه تکی سریال** با نمایش فصل‌ها و قسمت‌ها به صورت دسته‌بندی شده
- **پلیر ویدیوی آنلاین** با قابلیت پخش مستقیم (mp4, m3u8)
- **سیستم عضویت و پرداخت** با درگاه زرین‌پال (Sandbox و واقعی)
- **صفحه حساب کاربری** شامل: داشبورد، خرید عضویت، علاقه‌مندی‌ها، تاریخچه پرداخت، پروفایل
- **محدودیت دانلود** برای کاربران غیرعضو
- **لیست علاقه‌مندی** با قابلیت AJAX
- **فیلترینگ آرشیو** بر اساس ژانر و کشور
- **طراحی حرفه‌ای** با دارک مود، ریسپانسیو و انیمیشن
- **جستجوی اختصاصی** فقط در فیلم‌ها و سریال‌ها
- **وارد کردن دسته‌جمعی** قسمت‌های سریال
- **ACF فیلد گروپ‌ها** به صورت PHP (قابل انتقال بین سرورها)
- **IMDB** نمایش امتیاز و لینک
- **اسلاگ خودکار** از نام انگلیسی فیلم/سریال

## نیازمندی‌ها

- WordPress 6.0 یا بالاتر
- PHP 7.4 یا بالاتر
- افزونه Advanced Custom Fields Pro (برای فیلدهای سفارشی)
- مرچنت کد زرین‌پال (برای پرداخت آنلاین)

## نصب

1. پوشه `filmworld` را در `/wp-content/plugins/` آپلود کنید.
2. افزونه را از بخش "افزونه‌ها" فعال کنید.
3. فیلدهای ACF به صورت خودکار ثبت می‌شوند.
4. به **تنظیمات > مشخصات فیلم و سریال > تنظیمات پرداخت** بروید و مرچنت کد زرین‌پال را وارد کنید.
5. قوانین بازنویسی (Rewrite Rules) با فعال‌سازی افزونه به‌طور خودکار به‌روزرسانی می‌شوند.

## ساختار فایل‌ها

```
filmworld/
├── filmworld.php              # فایل اصلی پلاگین
├── uninstall.php              # پاک‌سازی هنگام حذف
├── assets/
│   ├── css/
│   │   ├── common.css         # استایل‌های مشترک
│   │   ├── dark.css           # دارک مود
│   │   ├── front-page.css     # صفحه اصلی
│   │   ├── archive.css        # آرشیو
│   │   ├── single.css         # صفحات تکی
│   │   ├── search.css         # جستجو
│   │   └── single-movie.css   # (Legacy)
│   └── js/
│       ├── main.js            # دارک مود، فاویوریت، انیمیشن
│       └── player.js          # پلیر ویدیو
├── includes/
│   ├── loader.php             # بارگذاری فایل‌ها
│   ├── post-types.php         # پست تایپ‌ها
│   ├── taxonomies.php         # تکسونومی‌ها
│   ├── acf-fields.php         # فیلدهای ACF
│   ├── template-loader.php    # قالب‌ها و استایل‌ها
│   ├── post-slug.php          # اسلاگ خودکار
│   ├── search.php             # فیلتر جستجو
│   ├── subscription.php       # بررسی عضویت
│   ├── payment.php            # درگاه زرین‌پال
│   ├── user-membership.php    # AJAX فاویوریت + حساب کاربری
│   ├── download-links.php     # پست تایپ لینک دانلود
│   ├── admin-menu.php         # منوی ادمین
│   ├── series-bulk-import*.php  # وارد کردن دسته‌جمعی
└── templates/
    ├── front-page.php
    ├── archive-movie.php
    ├── single-movie.php
    ├── single-series.php
    ├── search.php
    ├── searchform.php
    ├── account.php            # صفحه حساب کاربری
    └── parts/
        ├── media-card.php
        └── downloads.php
```
